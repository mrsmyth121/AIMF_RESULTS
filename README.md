# AIM‑F Results Repository

This repository stores **all evaluation outputs generated by AIM‑F**.  
Each directory represents a single model run and contains the full set of artifacts produced by the AIM‑F THETA pipeline.

The repository is **append‑only**: new results are added over time, and existing results are never modified.

---

## 📁 What Each Run Contains

A typical run folder includes:

- **checkpoint.complete.json** — Internal AIM‑F checkpoint metadata.
- **metrics.json** — Quantitative evaluation metrics for the run.
- **responses.json** — Full model responses for every prompt.
- **report.md** — Human‑readable summary of the run.
- **visualization_2d_dashboard.html** — Interactive 2D evaluation dashboard.
- **visualization_pca_category.html** — PCA visualization by category.
- **visualization_pca_strategy.html** — PCA visualization by strategy.
- **visualization_umap_category.html** — UMAP visualization by category.
- **visualization_umap_strategy.html** — UMAP visualization by strategy.

All `.html` files can be opened directly in any modern browser.
